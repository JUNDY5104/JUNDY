AMEBLO PROMPT GENERATOR

ChatGPT・Claude・Geminiなど、お好みのAIにそのまま貼り付けてお使いください。

このツールの使い方
3ステップで記事生成プロンプトが完成します
1

基本設定を入力する初回のみ

あなたのブログ名・アメブロID・体験談URLなど、ブログ固有の情報を入力します。入力内容はブラウザに自動保存されるので、次回以降は入力不要です。

↓
2

記事設定を入力する毎回

投稿予定日・テーマ・記事タイプなど、今回の記事に合わせた設定を入力します。項目を選ぶだけで完了します。

↓
3

プロンプトを生成してAIに貼り付ける

「プロンプトを生成する」ボタンを押すと、AIへの指示文が完成します。コピーしてChatGPT・Claude・GeminiなどのAIに貼り付けるだけで記事が生成されます。

ポイント：生成されたプロンプトは、アメブロの禁止タグ・HTMLルール・SEO対策・文体指定などをすべて自動で含んでいます。AIに貼り付けた後に追加の指示を出す必要はありません。

アメブロ禁止タグ一覧

以下のHTMLタグはアメブロでは使用できません。AIが生成した記事にこれらのタグが含まれていた場合は削除してください。

html head body frame frameset iframe object param script form input embed textarea meta button svg javascript server

※ option・title タグはフリースペース・自己紹介では禁止です

文字色・強調の書き方
用途 HTMLコード 表示例
自分の言葉（「○○○」） <b style="color:#00bcd4;">「○○○」</b> 「水色・太字」
相方の言葉（「○○○」） <b style="color:#ffb6c1;">「○○○」</b> 「薄ピンク・太字」
読者の言葉（「○○○」） <b style="color:#e91e8c;">「○○○」</b> 「濃いピンク・太字」
補足解説（※〇〇〇） <span style="color:#ff9800;">※○○○</span> 「オレンジ・通常」
強調（太字・赤） <b style="color:#cc0000;">○○○</b> 「赤・太字」
強調（赤＋下線） <b><u style="color:#cc0000;">○○○</u></b> 「赤・太字・下線」
見出しタグのコード

H2・H3タグはアメブロのデフォルトスタイルを上書きするため、以下のコードを使います。

H2見出し

<p></p><p></p><h2 style="color:#323232;font-size:20px;line-height:1.4;min-height:28px;padding:12px 16px;margin:8px 0;border-bottom:1px solid #898989;border-left:4px solid #898989;">見出しタイトル</h2>

H3見出し

<h3 style="position:relative;color:#333;font-weight:bold;font-size:20px;line-height:1.6;padding:4px 10px 4px 20px;margin:8px 0;border-bottom:2px solid #235fb0;"><span style="width:10px;height:100%;position:absolute;bottom:-2px;left:0;background-color:#235fb0;border-radius:2px;">&nbsp;</span><span>見出しタイトル</span></h3>
リブログカードの埋め込みコード

参考ブロガーの記事を紹介する際に使用します。[ブロガーID]と[記事ID]を差し替えてください。

<p>&nbsp;</p>
<p>※今回参照させていただいた記事は以下です。</p>
<p>
<iframe class="reblogCard" data-ameba-id="[ブロガーID]" data-entry-id="[記事ID]" frameborder="0" height="234px" sandbox="allow-same-origin allow-scripts allow-top-navigation" scrolling="no" src="https://ameblo.jp/s/embed/reblog-card/[ブロガーID]/entry-[記事ID].html?reblogAmebaId=あなたのID&amp;isLightPreview=true" width="100%"></iframe></p>
記事の基本フォーマット
ルール 内容
記事の書き出し <p>タグから始める（<html>・<body>タグは不要）
句点の後 必ず改行を入れる（句点で終わったら次の<p>タグへ）
行間の挿入 1行と1行の間には<p></p>タグを入れて行間を広げる
自分の言葉の前後 <p></p>を2つ入れて行間を広げる
相方・読者の言葉の前後 <p></p>を2つ入れて行間を広げる
X用ポスト 記事完成後にAIへ「X用のポストを8本、2時間おきの予約投稿用に作ってください」と追加依頼する
